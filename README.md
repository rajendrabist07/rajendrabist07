<br>
My work spans the full stack: from semantic, accessible HTML and scalable CSS architectures to JavaScript-driven interfaces and backend systems built with Node.js and MongoDB. I use frameworks like React and Next.js not as shortcuts, but as abstractions over principles I understand deeply.

**What I care about:**
*   Designing interfaces that scale, not just render
*   Writing code that remains understandable months later
*   Treating HTML and CSS as architectural decisions, not setup steps
*   Building systems where frontend and backend evolve cleanly together

I value **clean thinking** over noisy code, and **foundations** over trends.
